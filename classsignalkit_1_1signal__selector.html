Detailed Description

template<class TYPE, class MODULE>
class signalkit::signal_selector< TYPE, MODULE >

Signalkit selector signal. This class implements a TLM Signal abstraction of an outgoing signal with multiple channels. The signal stores the value and triggers the update function of all receivers. Each channel can be updated seperately. The channels are addressed as bitmasks. This makes it possible to update multiple channels at once.
