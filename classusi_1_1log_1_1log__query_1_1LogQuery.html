Detailed Description

Class for reading SoCRocket logs from HDF5-databases

Example:

import pandas as pd
import log_query as lq

log = lq.LogQuery("log.h5")
df = log.select(where="index >= -1", columns=['index', 'time', 'message_text'])
print df
log.close_file()
