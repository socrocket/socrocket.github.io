Detailed Description

template<class TYPE, class MODULE>
class signalkit::signal_base< TYPE, MODULE >

Base class for all signalkit signals. This is needed to hide the direct pointer to the parent model. Each instantiated member signal needs a pointer to it's parent class to provide an instance pointer for callback functions.

To provide these pointer we abuse the sc_object api. sc_object provides a mechanism to get an sc_object instance of the parent object. This is casted back to the type MODULE (template parameter).
